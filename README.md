# 网络安全学习笔记

这是一个使用 Obsidian 管理的网络安全学习笔记库。

## 目录结构

- `01 Link/` - 链接与资源
- `02 Sec/` - 网络安全相关笔记
  - `13 Tool/` - 工具使用笔记
- `03 Internship/` - 实习相关笔记

## 使用说明

本笔记库使用 [Obsidian](https://obsidian.md/) 进行编辑和管理。

### Git 同步

笔记通过 Git 进行版本控制，并定期自动同步到 GitHub。

### 自动同步脚本

仓库中包含一个自动同步脚本 `sync.sh`（或 `sync.ps1`），用于定期提交更改并推送到远程仓库。

## 注意事项

- 请勿提交敏感信息（如密码、密钥、个人数据等）。
- 定期备份重要笔记。
